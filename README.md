# American-Option-Pricing-Numerical-Finite-Diff-Schemes THESIS

## General information

This repository corresponds to my (Mariia Avdokhina's) thesis in partial fulfillment of the requirements of the degree of Bachelor in Data Science and Society at Central European University.

The practical part of this thesis focuses on implementing the forward Euler (explicit), backward Euler (implicit) and Rannacher finite difference schemes in Python, recording their scores on three different evaluation metrics (accuracy, stability, computational cost) and making a subsequent comparison and a conclusion based on this comparison. For an additional (but important) experiment, I built a simple PINN and set it against the reference solution (Rannacher on a fine grid).

In the thesis, I included Figure 3.1 "Payoff diagram of a call and a put option". I made that figure myself as well; the code for it is in a separate file.

## Repository structure

The repository is organized into three main folders: `src`, `experiments` and `results`.

### `src/`
This folder contains the core implementation of the finite difference methods used throughout the thesis.

- `grid.py`  
  Defines the spatial and temporal grids, model parameters and discretization settings.

- `payoff.py`  
  Contains the payoff functions for option pricing.

- `explicit.py`  
  Implements the explicit (forward Euler) finite difference scheme for the American put option.

- `implicit.py`  
  Implements the implicit (backward Euler) finite difference scheme using a banded matrix solver.

- `rannacher.py`  
  Implements the Rannacher scheme; combines initial backward Euler damping steps with a Crank–Nicolson phase.

- `metrics.py`   
  Contains functions used to evaluate numerical accuracy and compare solutions.

- `stability.py`
  An older file version of a stability evaluation for qualitative metrics.

### `experiments/`
This folder contains scripts used to run the numerical experiments presented in the thesis.

- `compare.py`  
  Generates the main five comparison plots between the explicit Euler, implicit Euler and Rannacher methods.

- `stability_compare.py`  
  Shows a case where explicit Euler is unstable and produces two plots.

- `payoff_diagrams.py`  
  Provides the code for Figure 3.1 in the thesis; created with artificial data for demonstration purposes only.

- `pinn_american_put.py`  
  Builds a simple feedforward PINN and compares its option price prediction against the refined benchmark solution obtained with the Rannacher method.

### `results/`
This folder stores the outputs generated by the experimental scripts.

- `figures/`  
  Contains the plots used in the thesis.

- `tables/`  
  Contains CSV tables with runtime, stability and accuracy results.

- `figs.py` and `stability_plot.py`  
  Helper scripts used for visualization and figure generation.

### Additional files

- `README.md`  
  Provides an overview of the repository and reproduction instructions.

- `requirements.txt`  
  Lists all Python dependencies required to reproduce the results.

- `.gitignore`  
  Excludes unnecessary files such as cache files and temporary outputs from version control.

- `archive folders`  
  Store the scripts and outputs of archived tests which were either recording intermediate results or rewritten because of mistakes in code.
