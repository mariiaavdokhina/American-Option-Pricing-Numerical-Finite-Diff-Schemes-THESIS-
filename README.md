# American-Option-Pricing-Numerical-Finite-Diff-Schemes THESIS

This repository corresponds to my (Mariia Avdokhina's) thesis in partial fulfillment of the requirements of the degree of Bachelor in Data Science and Society at Central European University.

All code provided in the repository is written by me. The data downloaded for this thesis is publicly accessible and free.

The practical part of this thesis focuses on implementing the forward Euler (explicit), backward Euler (implicit) and Rannacher time-stepping schemes in Python, recording their scores on three different evaluation metrics (accuracy, stability, computational cost), making a subsequent comparison and a conclusion based on this comparison.

In the thesis, I included Figure 3.1 "Payoff diagram of a call and a put option". I made that figure myself as well; the code for it is in a separate file.
